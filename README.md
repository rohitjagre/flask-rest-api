# flask-rest-api

Simple CRUD REST API using Flask, following best practices in mind.

## Run application
```python run.py```


## Run tests
``python -m pytest -v -s --cov=app``


## Generate coverage report from tests
``python -m pytest -v -s --cov=app --cov-report=html``
